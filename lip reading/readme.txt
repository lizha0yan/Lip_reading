This is a lip reading project. Using Neural Network to train the model. This model can recognize the viseme of mouth in videos. 

Use instruction:
1. Put your vedio files in the 'mov' folder.
2. Edit your input_path to the path you save the code.
3. Edit your result path(By defult is in result folder).


Notice: The accrucy is only 55%. You may need to get final results combining with other model's results.
